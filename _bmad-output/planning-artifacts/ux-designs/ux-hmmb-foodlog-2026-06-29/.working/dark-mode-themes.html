Nota de escopo: O PRD marca dark mode como non-goal para o MVP ("Não implementar dark mode nesta iteração"). Se você escolher uma variante dark aqui, o escopo do MVP se expande — o design system precisará de dois conjuntos de tokens e o CSS precisará de @media (prefers-color-scheme: dark). D4 (Neutral) é o mais simples de adicionar pois o fundo é zinc puro sem matiz azulado, tornando os overrides mínimos.
